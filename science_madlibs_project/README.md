# Science Mad Libs

An interactive Python program that allows people to create their own scientific
journal entries by filling in prompts(Mad Libs-style)

## Features
- Command line interface
- Customizable science journal templates
- Text generation based on user input
- Automatically generated abstract, introduction, methods, and results


## Technologies
- Python 3
- matplotlib

## Why I built this:
I built this project to practice Python functions, input validation, program
structure (main()), and basic data visualization with matplotlib

## How to run
```bash
python3 madlibs.py
